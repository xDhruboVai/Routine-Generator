# Routiner Khichuri

##https://routiner-khichuri.vercel.app/

## Run locally

### One command (recommended)
```bash
npm install
npm run dev
```

### Backend
```bash
cd backend
npm install
npm run dev
```

Runs at: http://localhost:4000

### Frontend
```bash
cd frontend
npm install
npm run dev
```

Runs at: http://localhost:5173
